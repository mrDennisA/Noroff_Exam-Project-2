## Getting Started

### Installing

1. Clone the repo:

```bash
git clone git@github.com:mrDennisA/Noroff_Exam-Project-2.git
```

2. Install the dependencies:

```
npm install
```

### Running

Run development server

```bash
npm run start
```

Build

```bash
npm run Build
```
